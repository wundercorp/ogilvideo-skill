# Ogilvideo Motion Direction Rubric

Builder Studio: https://builderstudio.dev

Use this rubric to evaluate or improve a motion video.

## Scoring scale

Score each category from 1 to 5.

- 1 means weak, unclear, generic, or harmful.
- 2 means direction exists but execution is inconsistent.
- 3 means solid and usable.
- 4 means polished and persuasive.
- 5 means campaign-level motion direction.

## Categories

| Category | What to evaluate |
| --- | --- |
| Strategic clarity | The audience, problem, promise, proof, and CTA are obvious. |
| Hook strength | The first 1 to 2 seconds earn attention. |
| Narrative arc | The video moves through problem, pressure, pivot, demo, proof, and CTA. |
| Motion meaning | Motion guides attention and supports the argument. |
| Visual cohesion | Palette, typography, cards, icons, depth, and backgrounds belong together. |
| Product believability | UI and data feel real, consistent, and trustworthy. |
| Typography readability | Text is short, legible, timed well, and hierarchically clear. |
| Pacing | The edit has rhythm, contrast, pauses, and acceleration. |
| Transition logic | Transitions express story logic instead of random effects. |
| Sound integration | Music and effects support timing and never overpower the message. |
| Platform readiness | Aspect ratio, safe areas, compression, captions, and CTA hold are ready. |
| Artifact control | No warped text, random UI, broken logos, unwanted overlays, or AI glitches. |

## Premium video threshold

A video should not be called premium unless it scores 4 or higher in these five areas:

1. Strategic clarity.
2. Typography readability.
3. Motion meaning.
4. Visual cohesion.
5. Product believability.

## Red flags

- The video looks good but the viewer cannot explain the product.
- The product UI is too blurry to trust.
- The animation has constant motion with no rests.
- Every transition uses a different style.
- Claims appear without proof or support.
- The logo is visible but the CTA is unclear.
- The final export includes device overlays or editing UI.

## Fix patterns

| Problem | Fix |
| --- | --- |
| Generic AI video feel | Lock styleframes, reduce prompt improvisation, composite UI manually. |
| Too much motion | Remove 30 percent of secondary movement, add holds before key claims. |
| Unreadable UI | Redesign UI at larger scale, blur only background panels, keep foreground crisp. |
| Weak hook | Replace generic intro with a painful truth, number, contrast, or provocative question. |
| Random transitions | Choose one motif and use it repeatedly with variation. |
| Confusing product flow | Add cursor, path, progress bar, map pin, or sequencing labels. |
| Cheap CTA | Remove extra decoration, center logo, hold for at least 2 seconds, add one clear action. |
